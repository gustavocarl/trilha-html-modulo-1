Entendendo a Comunicação Cliente x Servidor

Arquitetura Cliente-Servidor

É uma arquitetura que tem dois principais atores o Cliente e o Servidor sendo cada um tendo suas próprias responsabilidades, isso ajuda na manutenção e modularidade do sistema

Cliente

Cliente é onde o usuário interaje com as aplicações, ele apresenta a interface do usuário e envia as informações necessárias para o servidor. Exemplos de Clientes são navegadores web, softwares instalados e a tela visual de aplicações de celulares

Servidor

Servidor é onde as solicitações da aplicação são respondidas através de protocolos, nele são gerenciados os dados, a lógica do negócio e envia as respostas para o cliente. Servidores podem atender vários usuários simultaneamente, garantindo o desempenho e a estabilidade da aplicação.

Introdução ao HTML na Prática

Voltar

Estrutura HTML

A estrutura básica de uma página HTML é composta pelas tags <html> <head> <body>

  • <html> é a tag principal onde fica toda a página sendo parte visual ou não
  • <head> é a tag onde ficam informações não visuais onde que serão passadas para o navegador como arquivos css, fontes etc.
  • <body> é a tag que representa a parte visual do HTML, onde são mostrados paragráfos, imagens etc.

Tags HTML

As tags em HTML são elementos essenciais para a apresentação de uma página web. Cada tag possui uma função específica.

  • <u> tem a função de sublinhar um texto, como no exemplo a seguir: texto sublinhado.
  • <i> tem a função de deixar o texto em itálico: texto em itálico.
  • <sup> tem a função de exibir o texto em sobrescrito: texto sobrescrito.
  • <sub> tem a função de exibir o texto em subscrito: texto subscrito.

Textos

  • Em HTML, os textos podem ser escritos utilizando a tag <p>, que representa um parágrafo.

  • Para títulos, utilizam-se as tags de <h1> até <h6>, onde <h1> representa o nível mais importante e <h6> o menos relevante.

Links

  • Em HTML, existe a tag <a>, utilizada para criar links que conectam o usuário a outras partes do próprio site ou a páginas externas.
